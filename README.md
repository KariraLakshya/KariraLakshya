Stack: Gemini 2.5, E2B, Playwright, FastAPI, React, Redis.

**[Apex MLOps Auto-Trainer](https://github.com/KariraLakshya)** — Configuration-driven MLOps platform orchestrating model training, evaluation, and deployment via GitHub Actions with Docker containerization and data drift monitoring.
